What This App Does

The NYC Gas Price Tracker is a web app that pulls real weekly gas price data from the U.S. government's Energy Information Administration (EIA) and displays it in an easy to read format.

Because the EIA does not break gas prices down by NYC borough, this app uses New York State data the most local geographic level the EIA publishes. The app shows prices for regular, midgrade, premium, and diesel fuel.

  • Home page: Shows the latest weekly prices for New York State in a card + table layout
  • NY Prices by Area: Compares recent gas prices across multiple EIA-defined regions near New York
  • Documentation: This page — explains how everything works

🔌 What API Is Used?

This app uses the EIA Open Data API — a free, public API provided by the U.S. Energy Information Administration.

ProviderU.S. Energy Information Administration (EIA)
Base URLhttps://api.eia.gov/v2
Data FormatJSON
CostFree
Sign Uphttps://www.eia.gov/opendata/

Endpoints Used

Endpoint 1 — Weekly Gas Price Data
GET https://api.eia.gov/v2/petroleum/pri/gnd/data/ ?frequency=weekly &data[0]=value &facets[duoarea][]=SNY &sort[0][column]=period &sort[0][direction]=desc &length=20 &api_key=YOUR_API_KEY

This endpoint returns weekly retail gasoline and diesel prices. In this app, we filter it to New York State (SNY) and sort by most recent date first. The length=20 parameter limits results to the 20 most recent entries.

ParameterWhat it does
frequency=weeklyGet weekly data (not monthly or annual)
data[0]=valueInclude the price value in the response
facets[duoarea][]=SNYFilter to New York State only
sort[0][column]=periodSort by the date column
sort[0][direction]=descMost recent dates appear first
length=20Return only 20 rows
api_key=...Your free EIA API key
Endpoint 2 — Available Area Codes (Facets)
GET https://api.eia.gov/v2/petroleum/pri/gnd/facet/duoarea ?api_key=YOUR_API_KEY

This endpoint returns all valid geographic area codes (called duoarea) that exist in the gasoline prices dataset. We use this on the NY Prices by Area page to show users which areas the EIA tracks, and to fetch prices for each New York-relevant area.

API Key Setup

The EIA API requires a free API key. Without it, all requests will fail.
How to get your API key:
  1. Go to https://www.eia.gov/opendata/
  2. Click "Register" and fill out the short form
  3. Check your email for your API key
How to add your key to this app:

Open script.js and find this line near the top:

const API_KEY = "YOUR_API_KEY";

Replace YOUR_API_KEY with the actual key you received by email. For example:

const API_KEY = "abc123xyz456";
Important: Never share your API key publicly on GitHub or social media. If you host this on GitHub Pages, consider using an environment variable or backend proxy in a real project.
